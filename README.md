# VolcaModularMidi
This is a hardware mod to add MIDI IN capability to the [Korg Volca Modular](https://www.korg.com/us/products/dj/volca_modular/)


Bill Of Materials:
 * R1  220 ohm resistor
 * R2  270 ohm resistor
 * D1  1N4148 diode
 * J1  2.54mm header
 * J2  DIN-5 connector
 * U1  6n139 optocoupler
